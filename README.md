# WitchcraftForBeginners Design Package

Complete design system for WitchcraftForBeginners.com - the flagship spirituality site.

## Brand Identity
- **Voice**: Mystical warmth, accessible wisdom, grounded practitioner
- **Primary**: #4A1C6F (Deep Purple)
- **Secondary**: #C9A962 (Gold)
- **Background**: #0D0D0D (Near Black)
- **Display Font**: Cinzel
- **Body Font**: Lora

## Anti-Patterns
- NO Halloween imagery
- NO centered layouts
- NO sans-serif fonts
- NO purple-to-blue gradients
- NO bright saturated colors
